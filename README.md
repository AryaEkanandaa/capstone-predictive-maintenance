# Predictive Maintenance Copilot

## Ringkasan

Proyek **Predictive Maintenance Copilot** adalah *platform* berbasis AI untuk memprediksi kegagalan aset industri menggunakan data sensor.

Proyek ini dibagi menjadi tiga komponen utama:
* **`backend/`**: Node.js/Express API 
* **`frontend/`**: React Dashboard
* **`ML/`**: Python/Model Machine Learning
